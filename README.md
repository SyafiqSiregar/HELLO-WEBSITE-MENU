# Hello Coffee - Premium Menu

Proyek ini adalah digital menu untuk Hello Coffee, dirancang dengan estetika premium dan responsif untuk perangkat mobile.

## Struktur Folder

```text
hello-coffee/
│
├── index.html          # File utama (Struktur HTML)
│
├── css/
│   └── style.css       # Gaya kustom dan animasi
│
├── js/
│   ├── data.js         # Database produk/menu
│   └── script.js       # Logika interaksi dan rendering
│
└── assets/
    ├── logo.png        # Logo cafe (opsional)
    └── products/       # Tempat menyimpan gambar produk lokal
```

## Teknologi yang Digunakan
- **HTML5**
- **Tailwind CSS** (via CDN)
- **Vanilla JavaScript**
- **Google Fonts** (Inter & Playfair Display)
